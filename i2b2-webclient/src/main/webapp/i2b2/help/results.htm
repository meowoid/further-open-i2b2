Query Results

Results: Patients Common in All Sources

The number of patients that are common to all data sources. In the diagram below, patients which occur in the overlapped area are common to data source 1 and data source 2.

Results: Unique Patient Total

The number of individually unique patients from all data sources. If a patient is common to more than one data source, they are only counted once. In the diagram below, patients which occur in the overlapped area of data source 1 and data source 2 are only counted once.

Results: Sum

The sum of aggregate counts received from all data sources. If a patient is common to more than one data source, they are counted once for each data source. In the diagram below, everything from data source 1 and data source 2 is included, including the area for which they overlap.
