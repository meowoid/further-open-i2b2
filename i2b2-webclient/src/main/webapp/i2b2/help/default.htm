Introduction

i2b2 (Informatics for Integrating Biology and the Bedside) is an NIH-funded National Center for Biomedical Computing. The i2b2 Center is developing a scalable informatics framework that will bridge clinical research data and the vast data banks arising from basic science research in order to better understand the genetic bases of complex diseases. This knowledge will facilitate the design of targeted therapies for individual patients with diseases having genetic origins. The i2b2 web client is a web-based interface to the i2b2 repository. It has been modified for FURTHeR to connect to the FURTHeR federated query engine in order to allow queries to span multiple, heterogeneous data sources.

Browser Requirements

The i2b2 web client has been tested on Internet Explorer 6 and 7, Firefox 2 and 3, Safari 3, and Chrome 15. It works in both Windows and Mac OS. JavaScript must be enabled.

Analysis Tools

The i2b2 web client contains an Analysis Tools page that provides plugins that perform a variety of data analysis and visualization functions. The Plugins view lists the available plugins for your project. FURTHeR currently supports only a Demographics analysis tool, which allows you to view demographic feature breakdowns (sex, age, etc.) for the result cohort. Click a plugin to load it into the Plugin Viewer. Some plugins contain additional help information for using that plugin. See the Results Analysis video for additional help on using the Demographics plugin.
